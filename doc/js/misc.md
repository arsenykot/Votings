### `b64enc(str) -> String`
Кодирует строку в Base64. В отличие от встроенного `btoa`, поддерживает кириллицу
|Параметр|Тип     |Описание                       |
|:-------|:-------|:------------------------------|
|`str`   |`String`|Строка для кодирования в Base64|



### `b64dec(b64str) -> String`
Декодирует строку из Base64. В отличие от встроенного `atob`, поддерживает кириллицу
|Параметр|Тип     |Описание                          |
|:-------|:-------|:---------------------------------|
|`b64str`|`String`|Строка для декодирования из Base64|
