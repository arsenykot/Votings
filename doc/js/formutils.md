# Утилиты для работы с формами
[Выше](/doc/js/index.md) | [Корень](/doc/index.md)

Предоставляет методы-утилиты для работы с формами. Функционал реализован в `/main/static/util.js` в регионе `formutils`

### `validateForm(form) -> bool`
Проверяет, валидны ли данные, введённые в форму
|Параметр|Тип              |Описание|
|:-------|:----------------|:-------|
|`form`  |`HTMLFormElement`|Форма   |



### `validateFormInput(input) -> bool`
Проверяет, валидны ли данные, введённые в поле ввода
|Параметр|Тип               |Описание  |
|:-------|:-----------------|:---------|
|`input` |`HTMLInputElement`|Поле ввода|
