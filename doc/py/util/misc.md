### `@check_auth(auth, bans, redir) -> Func`
Декоратор для быстрой проверки прав доступа
|Параметр|Тип    |Описание                                                                                   |
|:-------|:------|:------------------------------------------------------------------------------------------|
|`auth`  |`bool?`|Проверять ли то, вошёл ли пользователь в систему? По умолчанию - True                      |
|`bans`  |`bool?`|Проверять ли то, заблокирован ли аккаунт пользователя? По умолчанию - True                 |
|`redir` |`bool?`|Переадресовать(True) или вернуть API-ответ(False) при провале проверок. По умолчанию - True|
