### `<docs>`
Корневой элемент документации. Модифицировать прямо дочерние параметры не рекомендуется
|Параметр  |Тип        |Описание                                             |
|:---------|:----------|:----------------------------------------------------|
|`path`    |`xs:string`|Смещение пути, по которому расположена документация  |
|`title`   |`xs:string`|Заголовок индекса                                    |
|`desc?`   |`xs:string`|Описание индекса. Необязательный аргумент            |
|`folder*?`|`folder`   |Папка с документацией (необязательно, мульти-элемент)|



### `<folder>`
Папка с документацией
|Параметр  |Тип        |Описание                                                               |
|:---------|:----------|:----------------------------------------------------------------------|
|`path`    |`xs:string`|Смещение папки относительно родительского элемента (aka название папки)|
|`title`   |`xs:string`|Заголовок индекса                                                      |
|`desc?`   |`xs:string`|Описание этой папки с документацией (необязательно)                    |
|`file*?`  |`file`     |Файл документации (необязательно, мульти-элемент)                      |
|`folder*?`|`xs:string`|Подпапка с документацией (необязательно, мульти-элемент)               |



### `<file>`
Файл документации
|Параметр  |Тип        |Описание                               |
|:---------|:----------|:--------------------------------------|
|`path`    |`xs:string`|Название файла документации            |
|`title`   |`xs:string`|Заголовок файла                        |
|`desc?`   |`xs:string`|Описание этого файла (необязательно)   |
|`method*?`|`method`   |Метод (необязательно, мульти-элемент)  |
|`xmltag*?`|`xmltag`   |XML-тег (необязательно, мульти-элемент)|
|`table*?` |`table`    |Таблица (необязательно, мульти-элемент)|
