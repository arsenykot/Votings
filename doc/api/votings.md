# Эндпоинты, связанные с голосованиями
[Выше](/doc/api/index.md) | [Корень](/doc/index.md)



### `/api/votings/new?name&description&option1&option2&close&date&time` (POST)

Создаёт голосование

#### Требуемые аргументы:
|Аргумент     |Тип                 |Описание                                                 |
|:------------|:-------------------|:--------------------------------------------------------|
|`name`       |`String[1..20]`     |Название голосования                                     |
|`description`|`String[1..100]?`   |Описание голосования                                     |
|`option1`    |`String[1..20]`     |Опция 1 (в дальнейшем будет переработано)                |
|`option2`    |`String[1..20]`     |Опция 2 (в дальнейшем будет переработано)                |
|`close`      |`"auto" \| "manual"`|Способ закрытия голосования                              |
|`date`       |`String?`           |Дата автоматического закрытия. Только при `close = auto` |
|`time`       |`String?`           |Время автоматического закрытия. Только при `close = auto`|



#### Эндпоинт возвращает:
|Статус|Данные        |Когда                                                               |
|:-----|:-------------|:-------------------------------------------------------------------|
|`400` |`BADREQUEST`  |Передано недостаточно аргументов или аргументы имеют неверный формат|
|`400` |`BADDATE`     |Дата или время имеют неверный формат                                |
|`401` |`UNAUTHORIZED`|Пользователь не авторизован                                         |
|`200` |`OK`          |Голосование успешно создано                                         |
