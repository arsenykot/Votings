# Эндпоинты, связанные с регистрацией
[Выше](/doc/api/index.md) | [Корень](/doc/index.md)



### `/api/registration/check_name?username` (GET)

Проверяет, занято ли указанное имя

#### Требуемые аргументы:
|Аргумент  |Тип  |Описание        |
|:---------|:----|:---------------|
|`username`|`str`|Имя пользователя|



#### Эндпоинт возвращает:
|Статус|Данные      |Когда                                         |
|:-----|:-----------|:---------------------------------------------|
|`400` |`BADREQUEST`|Не указано имя пользователя                   |
|`400` |`REGEX`     |Имя пользователя содержит недопустимые символы|
|`200` |`OK`        |Указанное имя свободно                        |
|`200` |`TAKEN`     |Указанное имя занято                          |



### `/api/registration/register?username&password` (POST)

Создаёт аккаунт с указанным именем пользователя и паролем, затем авторизуется в него

#### Требуемые аргументы:
|Аргумент  |Тип         |Описание        |
|:---------|:-----------|:---------------|
|`username`|`str[4..24]`|Имя пользователя|
|`password`|`str[8..]`  |Пароль          |



#### Эндпоинт возвращает:
|Статус|Данные         |Когда                                                     |
|:-----|:--------------|:---------------------------------------------------------|
|`400` |`BADREQUEST`   |Недостаточно аргументов или формат аргументов некорректный|
|`400` |`REGEX`        |Имя пользователя содержит недопустимые символы            |
|`400` |`USERNAMETAKEN`|Имя пользователя занято                                   |
|`200` |`OK`           |Аккаунт создан, авторизация выполнена                     |
