# Эндпоинты, связанные с автозирацией
[Выше](/doc/api/index.md) | [Корень](/doc/index.md)



### `/api/auth/login?username&password` (POST)

Попытаться войти в систему с указанным именем пользователя и паролем

#### Требуемые аргументы:
|Аргумент  |Тип  |Описание        |
|:---------|:----|:---------------|
|`username`|`str`|Имя пользователя|
|`password`|`str`|Пароль          |



#### Эндпоинт возвращает:
|Статус|Данные          |Когда                                             |
|:-----|:---------------|:-------------------------------------------------|
|`400` |`BADREQUEST`    |Недостаточно аргументов                           |
|`403` |`BADCREDENTIALS`|Переданные имя пользователя и / или пароль неверны|
|`200` |`OK`            |Авторизация прошла успешно                        |



### `/api/auth/logout` (GET / POST)

Выйти из аккаунта пользователя и перейти на главную страницу
