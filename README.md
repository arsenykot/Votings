# Simple Votings
## Подготовка к работе
- Убедитесь, что у Вас установлена библиотека django. Тестирование работы проекта проводилось на версии django 6.0

- Произведите миграцию базы данных:
```
python manage.py makemigrations
python manage.py migrate
```

- Создайте аккаунт Администратора:
```
python manage.py createsuperuser
```

- Запустите сервер:
```
python manage.py runserver
```
